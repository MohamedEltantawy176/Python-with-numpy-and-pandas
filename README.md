# Python Project Setup

This project uses **Python 3.12**, along with **NumPy** and **Pandas**.  
Follow the steps below to set up your environment and run the project.

---

## Prerequisites

- Install **Anaconda** locally.
- Ensure Python 3.12 is available.

---

## Step 1: Create a Conda Environment

Create a new environment named `venv` with Python 3.12:

-> conda create -p venv python==3.12

## Step 2: Install Dependencies

-> pip install -r requirements.txt
